                          # 前言

                          作为一个程序员，可能更多的是在 Linux 中使用命令来操作。但在日常使用 Windows 的过程中，或多或少会使用到命令提示符窗口，也就是 Windows 中的 CMD。这个时候，掌握一些常用的命令就尤为重要了，一方面方便自己使用，另一方面，也方便给电脑小白露一手 👀。那接下来就来看看，CMD 中最常用的一些命令。

                          # 如何打开命令提示符窗口

                          打开的方式很多，这里例举常用的 5 种。

                          1. 方法 1

                          Win + R 快捷键打开运行，输入 cmd 后回车。

                          1. 方法 2

                          依次打开开始菜单 -> Windows 系统（Windows 工具） -> 命令提示符。

                          1. 方法 3

                          打开资源管理器，然后在地址栏输入 cmd 后回车。

                          1. 方法 4

                          鼠标右键点击开始菜单的 Win 图标，找到命令提示符（终端）并进入。

                          1. 方法 5

                          在任务栏中的搜索栏中搜索 cmd 并打开。

                          # 常用命令

                          # 目录操作

                          1. 磁盘切换
                          盘符 + 冒号

                          如 D: 就代码进入 D 盘。

                          1. 进入指定目录
                          cd 目录

                          1. 返回磁盘根目录
                          cd \

                          1. 返回上级目录
                          cd ..

                          ![](./assets/20221126-cmd-command/cd up.png)

                          1. 新建目录
                          md 目录名

                          1. 查看当前目录中内容
                          dir

                          1. 查看指定目录结构
                          tree 目录

                          1. 删除目录
                          rd 目录名

                          1. 复制文件
                          copy 源文件 目标文件

                          1. 删除文件
                          del 文件名

                          # 开关机操作

                          1. 关机
                          shutdown /s
                          1. 重启
                          shutdown /r
                          1. 注销
                          shutdown /l
                          1. 休眠
                          shutdown /h /f
                          1. 定时关机
                          shutdown /s /t 时间（秒）

                          1. 取消关机
                          shutdown /a

                          # 网络操作

                          1. 测试网络延迟和丢包率
                          ping ip/域名

                          1. 查看本机 ip 地址
                          ipconfig

                          1. 清除本地 DNS 缓存
                          ipconfig /flushdns

                          1. 路由追踪
                          tracert ip/域名

                          # 进程/服务操作

                          1. 查看当前运行的进程
                          tasklist

                          1. 运行指定程序或命令
                          start 程序名
                          1. 按名称结束进程
                          taskkill /im 进程名

                          1. 按 PID 结束进程
                          taskkill /pid PID

                          1. 查看当前运行的服务
                          net start

                          1. 启动指定服务
                          net start 服务名

                          1. 停止指定服务
                          net stop 服务名

                          # 其他操作

                          1. 清除屏幕
                          cls
                          1. 退出
                          exit

                          # 总结

                          以上就是今天所带来的 Windows 中常用的 CMD 命令了，对付日常使用完全够了，更多的一些命令用到的频率较低，可以在有需要的时候再去具体搜索。
