什么是 Nginx

Nginx 是一个跨平台、轻量级、高性能的 HTTP 和反向代理 web 服务器，而且同时也提供了 IMAP/POP3/SMTP 服务。Nginx 最高能够支持高到 50000 个并发连接数的相应，在最高并发连接的情况下，可以作为 Apache 服务的一个替代品。
